Simple Header Footer HTML
===============================
[![Build Status](https://travis-ci.org/Seravo/wp-simple-header-footer-html.svg?branch=master)](https://travis-ci.org/Seravo/wp-simple-header-footer-html)

A simple plugin for injecting HTML into various places in your Wordpress theme output.

Available for installation via [Composer via Packagist](https://packagist.org/packages/seravo/wp-simple-header-footer-html) or [GitHub Plugin Search](https://github.com/brainstormmedia/github-plugin-search).

For more information see **readme.txt**.
